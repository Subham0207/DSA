# Important Graph DSA Templates
- Dijkstra's Algorithm: Shortest path in weighted graph ( Min priority queue ).
- Topological Sort ( Kahn Algorithm / DFS): For Dependency and scheduling problems.
- Disjoint Set Union ( DSU)/ Union Find: Crucial for dynamic connectivity and Kruskal's minimum spanning tree.
- Tarjan's / Kosaraju's: Finding strongly connected Components ( SCCs ).
